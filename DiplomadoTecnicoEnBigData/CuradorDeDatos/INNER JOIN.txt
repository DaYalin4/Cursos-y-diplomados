
  INNER JOIN 
    Sintaxis básica: 

      SELECT tabla1.columnas 
      FROM tabla1
      INNER JOIN tabla2 ON tabla1.columna = tabla2.columna

   Sintaxis con 3 tablas (Mejor manera para mantenimiento): 

      SELECT 
          c.nombre AS cliente,
          p.fecha_pedido,
          pr.nombre_producto,
          pr.precio,
          pr.cantidad,
          p.total
      FROM 
          Clientes c
      INNER JOIN 
          Pedidos p ON c.id = p.cliente_id
      INNER JOIN 
          Productos pr ON p.id = pr.pedido_id
      ORDER BY 
          p.fecha_pedido;

Otra forma: 
      SELECT c.nombre, p.fecha_pedido, pr.nombre_producto
      FROM Clientes c, Pedidos p, Productos pr
      WHERE c.id = p.cliente_id
      AND p.id = pr.pedido_id

  Si queremos ver los pedidos con los nombres de los clientes:

SELECT Clientes.nombre, Pedidos.producto, Pedidos.cantidad
FROM Clientes
INNER JOIN Pedidos ON Clientes.id = Pedidos.cliente_id
